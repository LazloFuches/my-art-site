robertegert.com is a "static" website: an archive of art projects, a biography, an ephemera section, and a contact form. 
Robert Egert is a visual artistwith an exhibition history primarily in New York City.
"Static" means the website is a set of pre-built files that are simply handed to visitors - there is no database and no software running on a server answering each visit. 
This makes it fast, secure, and nearly maintenance-free.
